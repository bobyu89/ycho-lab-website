# 智慧醫療轉譯及創新實驗室 — 網站改版設計文件

日期：2026-07-06
狀態：待使用者審查
現行網站：https://ycho-smart-health-lab.web.app/（原始碼備份已存於 scratchpad：site.html / styles.css / script.js）

## 1. 目標

把現有單頁靜態網站改版為**多頁式**實驗室網站，首頁為視覺重點（landing page），風格定位為「**霧灰科技感 + 液態玻璃 + 微動畫**」，同時讓老師能透過 Google 試算表與 Google 雲端硬碟自行更新消息與照片，不需碰程式碼。

### 非目標（這版不做）
- 中英全站切換（原網站有，這版拿掉；區塊標籤保留英文小標點綴國際感）
- 後台管理介面／會員系統
- 部落格／文章系統

## 2. 技術方案（已確認：方案 A）

- **純靜態網站**：HTML + CSS + 原生 JS，無框架、無建置工具，延續 Firebase Hosting 部署
- **共用頁首頁尾**：以 JS 元件注入（`components.js` 定義 header/footer 模板，各頁引用），實際只維護一份
- **最新消息／重要活動**：讀取公開發布的 Google 試算表（gviz JSON 端點，不依賴第三方服務）
- **活動照片**：讀取設為「知道連結者可檢視」的 Google 雲端資料夾（Drive API v3 `files.list` + API 金鑰，縮圖用 `drive.google.com/thumbnail?id=...&sz=w800`）
- **降級策略**：任一外部資料源讀取失敗時，顯示內建的靜態備用內容（寫死在 JS 的最後已知資料），版面不破
- 老師姓名在論文列表中粗體（Ho YC／賀彥中）；老師介紹區的重要頭銜、學歷、專長關鍵字粗體

## 5. 視覺系統

### 配色（已確認：灰主體 + 青紫點綴）

| 角色 | 色票 | 用途 |
|---|---|---|
| 底色 | #F4F5F7 淺霧灰 | 頁面背景 |
| 表面 | #FFFFFF | 卡片內部、玻璃基底 |
| 主文字 | #1E2427 炭黑 | 標題、內文 |
| 次文字／主調 | #8E8E8E 中灰 | 輔助文字、邊框、mono 標籤 |
| 點綴 1 | #00E3E3 螢光青 | 發光元素限定：波形、光暈、hover 光效、漸層、裝飾線（不做內文/按鈕色） |
| 點綴 1 深 | #009E9E | 需要可讀性的青色場合：連結、圖示 |
| 點綴 2 | #6A6AFF 靛紫 | 主按鈕、分類籤、漸層搭檔 |
| 光暈漸層 | 青 #00E3E3 → 紫 #6A6AFF（低飽和大模糊） | 玻璃元素背後的流動色彩 |

無障礙原則:#00E3E3 對比不足，僅作裝飾發光體；文字與互動元件用 #6A6AFF、#009E9E、#1E2427，均達 WCAG AA。

### 字體
- 中文／內文：Noto Sans TC（400/500/700/900）
- 英文標題：Inter（沿用現網站）
- 終端機標籤：Space Mono（`[EMA.MONITOR.ACTIVE]`、`PI: HO YEN-CHUNG`、統計卡標籤等）

### 液態玻璃系統（參考稿元素 + 前輪定案）
統一 `.glass` 樣式：半透明白 + `backdrop-filter: blur(12–16px) saturate(1.5)` + 1px 半透明白邊框；hover 時 blur 加深、邊框變亮、青色柔光陰影。
應用位置（克制原則，只用在浮於內容上方的元素）：
1. 膠囊導覽列（全站最明顯的玻璃時刻）
2. Hero 監測卡片（含 4–5 秒週期斜向光澤掃過）
3. 統計卡 ×4
4. 研究主軸卡
5. 照片 `Fig.` 小籤與 hover 說明條
6. 回到頂部按鈕、消息分類籤
降級：不支援 `backdrop-filter` 時自動改半透明白底實色。

### 微動畫清單
| 位置 | 動畫 |
|---|---|
| Hero 背景 | 青紫光暈漂移 + 呼吸縮放（15s 級慢速） |
| Hero 波形 | EMA 曲線無限橫向流動 + 移動平均虛線 |
| 玻璃卡片 | 斜向光澤掃過（sheen，4–5s 週期） |
| 導覽列 | 捲動時玻璃加深、輕微縮小 |
| 全區塊 | 捲動進場淡入上滑，卡片群錯開 0.1s（IntersectionObserver） |
| 主軸卡 | hover 浮起 + 頂部青→紫漸層線由左展開 + 編號變色 |
| 消息時間軸 | 豎線由上而下畫出、節點依序點亮 |
| 統計數字 | 進場時 0 滾動到目標值 |
| 照片牆 | hover 微放大 + 玻璃說明條滑入；燈箱彈性緩動 |
| 按鈕 | hover 微放大 + 箭頭滑動；主按鈕柔和脈動光暈 |
| 頁面切換 | 整頁淡入 |

全部包在 `prefers-reduced-motion` 檢查內，使用者系統設定減少動態時自動停用。

## 6. Google 後台資料架構

### 試算表（一份，兩個工作表，發布到網路）
- **news**：`date`（YYYY-MM-DD）、`category`（榮譽｜發表｜活動｜公告）、`content`、`link`（選填）
- **events**：`date`、`title`、`description`、`image`（選填，圖片網址）、`pinned`（TRUE 顯示於首頁重要活動區）

讀取方式：`https://docs.google.com/spreadsheets/d/{ID}/gviz/tq?tqx=out:json&sheet={name}`，前端解析。依日期新→舊排序。

### 雲端資料夾（活動照片）
- 資料夾權限「知道連結的人可檢視」
- Drive API v3 `files.list?q='{folderId}' in parents and mimeType contains 'image/'`（需 API 金鑰，佈署前由使用者在 Google Cloud Console 建立，金鑰限制 referrer 為網站網域）
- 縮圖：`https://drive.google.com/thumbnail?id={fileId}&sz=w800`；檔名（去副檔名）作為照片說明
- 首頁取最新 8–10 張；gallery.html 列全部

### 交付時附「維護說明」文件
一頁 README 教老師：怎麼在試算表加消息、怎麼上傳照片、檔名命名建議（例：`2026-06-15 EAFONS 研討會合影.jpg`）。

## 8. 測試與驗收

- 桌機（1280+）／平板（768）／手機（375）三尺寸版面檢查（Preview 工具實測）

- `prefers-reduced-motion` 開啟時動畫停用
- 不支援 backdrop-filter 的降級樣式
- Lighthouse：效能與無障礙 ≥ 90
- 內容比對：現網站所有文案（四主軸、4 計畫、2 研究生、12 篇論文、聯絡資訊）無遺漏
